智能电表能效分析智能体
=======================

一、这是什么？

这是一个 AI 智能体，可以自动分析你提供的电表数据文件（CSV 或 Excel 格式）。  
你只需要把文件发给它，它就会告诉你：

  - 数据的基本情况（总记录数、时间范围）
  - 每天的用电量统计（日均用电、最高日）
  - 能效评估（综合评分和等级，比如“良好”或“较差”）
  - 异常检测（有没有不正常的用电情况，严重程度如何）
  - 一张功率趋势图，让你直观地看到用电变化

它就像一个懂电的助手，把枯燥的数据变成清晰易懂的报告。


二、有什么用？

1. 家庭用户
   - 看看这个月用电是不是比上个月多了。
   - 检查有没有半夜突然耗电很高，是不是什么电器忘记关了。
   - 了解自己家的能效水平，找到省电空间。

2. 商铺或小企业
   - 分析每天营业用电和闭店用电是否合理。
   - 及时发现设备异常（比如某个机器一直在空转浪费电）。

3. 学习与研究
   - 如果你是电气、自动化或能源专业的学生，可以用它来快速分析实验数据。
   - 也可以把它当作“智能体开发”的案例，了解如何把 Python 程序变成 AI 助手。


三、怎么用？

使用方式很简单，分两步：

第一步：准备数据
  你的电表数据需要是 CSV 或 Excel 文件。
  文件中必须包含以下几列（列名要完全一致）：
    timestamp     - 时间（格式：2024-03-15 12:00:00）
    voltage       - 电压（伏特）
    current       - 电流（安培）
    active_power  - 有功功率（瓦特）

  示例数据（CSV 格式）：
    timestamp,voltage,current,active_power
    2024-03-15 00:00:00,220.5,2.30,500.0
    2024-03-15 00:01:00,221.0,2.31,502.0
    ...

  如果你有测试数据文件，直接用它就行（项目中 assets 文件夹里有一些示例）。

第二步：和智能体对话
  1. 打开智能体的对话界面（目前部署在 Coze 平台上，可以通过 Coze 的 Bot 聊天窗口使用）。
  2. 在聊天框里上传你的 CSV 或 Excel 文件。
  3. 发送“帮我分析一下”或者直接上传文件即可。
  4. 智能体会自动处理，几秒钟后返回一份分析报告和一张功率趋势图。

  你也可以追问，比如：
    - “哪天的用电量最高？”
    - “异常情况具体是怎么回事？”
    - “能效评分低的原因是什么？”

  智能体会根据分析结果进一步回答。


四、报告怎么看？

报告包含以下几个部分：

  📊 数据摘要
    - 总记录数：文件里有多少条数据
    - 时间范围：数据从什么时候开始，到什么时候结束

  📈 日用电分析
    - 日均用电：每天平均用多少度电（kWh）
    - 最高日用电：哪一天用电最多，用了多少

  ⚡ 能效评估
    - 综合评分：0-100 分，分数越高越好
    - 能效等级：根据评分划分为“优秀”“良好”“一般”“较差”
    - 提示：如果分数低于 70，可能设备老化或有待机耗电过多

  ⚠️ 异常检测
    - 告警级别：正常 / 注意 / 警告 / 严重
    - 异常总数和高优先级异常数
    - 如果存在异常，会列出部分异常发生的时间和功率

  📈 功率趋势图
    - 一张时间-功率的折线图，让你一眼看出用电波动
    - 高峰、低谷、突变都一目了然


五、常见问题

Q: 文件格式不对怎么办？
A: 确保文件是 CSV 或 Excel 格式，且包含 timestamp、voltage、current、active_power 这四列。列名大小写要一致。

Q: 上传后没有反应？
A: 如果文件很大，可能需要稍等几秒。如果长时间无响应，检查文件是否损坏，或者联系开发者查看日志。

Q: 能效评分为什么是 0 分？
A: 通常是因为数据中缺少电压或电流列，或者数据全部一样，导致计算负载率时分母为零。请检查数据列是否齐全。

Q: 我不想用 Coze，可以自己部署吗？
A: 可以。项目代码开源，你可以按照 README_DEV.md（如果有）中的说明，在自己的服务器上搭建 API 服务，然后对接任何前端或聊天平台。
